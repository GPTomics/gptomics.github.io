Palette comparison — current vs iris dichroic

Left: the live editorial single-warm palette (forest green brand + terracotta editorial accent). Right: the proposed iris triadic palette — iris violet anchors as brand (cool gravitas), warm red carries text accents (links, dates, like the current terracotta) for legible AA contrast, electric cyan returns as a decoration-only spark (blockquote rule, highlighted phrase) where contrast doesn't matter. Three roles, each suited to its perceptual physics: warm hues for text, cool cyan for illumination, violet bridging them.

Research note

Token economics of single-cell foundation models

By Domen Jemec · 2026-04-22

Training a foundation model on cell atlases is a different beast than language. The cost curve bends earlier because the data ceiling is much closer — a few million well-annotated cells, not a few trillion tokens.

The cheapest cell is the one you don't sequence twice.

Below: a per-tissue summary in editorial style. Inline code looks like scanpy.read_h5ad(), and a code block:

# clip outlier cells before embedding
import scanpy as sc
adata = sc.read_h5ad('atlas.h5ad')
